各种排序总结（JAVA）

冒泡排序（正序）

原理：每次循环都将前一位数和后一位数进行比较，并将大的数字放在后面。

时间复杂度 空间复杂度
最好 最坏
O(n) O(n^2) O(1)

基础实现：

优化算法：若某一次循环没有发生变化，则终止循环

快速排序

原理：通过一趟排序将要排序的数据分割成独立的两部分，其中一部分的所有数据都比另外一部分的所有数据都要小，然后再按此方法对这两部分数据分别进行快速排序，整个排序过程可以递归进行，以此实现整个数据变成有序序列。

时间复杂度 空间复杂度
最好 最坏
O(nlogn) O(n^2) O(1)
